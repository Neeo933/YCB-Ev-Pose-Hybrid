### 2. 核心架构：引导式双流融合网络 (Guided-Dual-Flow Network)

我们将之前的“两阶段检测器”升级为**“物理启发式融合架构”**。其核心逻辑不是 MTS + Depth，而是 **Depth \xrightarrow{Guided \ by} MTS**。

#### 创新点 1：MTS 驱动的边缘锐化 (MTS-Edge Refinement)

* **逻辑**：深度相机在物体交界处（尤其是杂乱堆叠时）会产生“深度撕裂”或边缘模糊。MTS 的彩色条纹本质上是物体的“动态几何指纹”。
* **实现**：提取 MTS 的高通特征（High-pass features），作为掩码（Mask）作用于深度图，强行校正深度图在物体边缘的响应。

#### 创新点 2：非对称交叉注意力融合 (Asymmetric Cross-Attention)

* **逻辑**：不使用对称融合。设计一个非对称模块：
* **Query (查询)** 来自 Depth：确定物体的基本几何结构（“这是什么形状？”）。
* **Key/Value (键值)** 来自 MTS：提供高频时空约束（“边缘在哪里动？”）。


* **学术价值**：这体现了“因果律”——空间结构由深度定，运动边界由事件定。

### 3. 分步实践指南 (Implementation Roadmap)

#### 第一阶段：物理对齐与可视化验证 (本周核心)

教授建议的“梯度图相乘叠显”是神来之笔。这是你论文中 **Figure 1 (Motivation)** 的来源。

* **动作**：
1. 计算 MTS 的空间梯度 G_{mts}。
2. 计算 Depth 的空间梯度 G_{depth}。
3. 将两者叠加：G_{combined} = G_{depth} \odot G_{mts}（或加权叠加）。


* **预期效果**：你会发现 G_{mts} 像一把手术刀，切开了 G_{depth} 中原本模糊不清的堆叠边缘。

#### 第二阶段：引导式特征提取模块开发 (第 2-4 周)

不再修改整个 Backnone，而是设计一个特殊的 **Guided-Header**。

* **代码实现**：
```python
# 伪代码：MTS 引导深度特征
feat_mts = backbone_mts(mts_img)     # 提取时间指纹特征
feat_depth = backbone_depth(depth_img) # 提取空间先验特征

# 生成空间注意力图
attention_map = torch.sigmoid(conv_layer(feat_mts))

# 引导融合：用 MTS 锐化深度特征
refined_depth_feat = feat_depth * (1 + attention_map) 

```


* **模型选择**：第一阶段检测器建议使用轻量级的 **YOLOv8-Pose**，第二阶段位姿解算使用 **GDR-Net** 风格的密集对应关系预测。

#### 第三阶段：针对“极端遮挡”的压力测试 (第 5-6 周)

* **实验设计**：在测试集中人为挑选遮挡比例（Occlusion Rate）为 30%、50%、70% 的样本。
